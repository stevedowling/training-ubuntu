# Module 2 · Users & Permissions

> **Goal:** understand who can do what on a Linux system - read any `ls -l` line at a glance, set ownership and permissions deliberately, use sudo with confidence, and know what an inode is.

## Learning objectives

- Explain users, groups, and root, and read `/etc/passwd` and `/etc/group`
- Decode and set permissions with `chmod` (symbolic and octal) and `chown`
- Use `sudo` (sudo-rs on 26.04) safely, and know why you don't log in as root
- Understand special bits: setuid, setgid, sticky - and why `/tmp` needs one
- Explain inodes, and the difference between hard links and symlinks
- Create and switch users and groups: `adduser`, `usermod`, `su`, `groups`

## Planned lessons

1. Users, groups, and root - who's who on the system
2. Reading and setting permissions - rwx, chmod, chown, umask
3. sudo - borrowing root safely
4. Links and inodes - what a file really is
