# Module 1 · First Contact

> **Goal:** get comfortable in a terminal - know what you're actually talking to when you type a command, move around the filesystem confidently, manage files, and find answers without leaving the shell.

Everything else in this course builds on these four lessons. If you already live in a terminal, skim lesson 1 anyway - it covers what changed in Ubuntu 26.04 - then jump ahead via the [course map](../README.md).

## Lessons

| # | Lesson | You'll learn |
|---|--------|--------------|
| 1 | [What Linux actually is](01-what-linux-actually-is.md) | Tell the kernel, the shell, and the distribution apart - and identify each one on your own machine |
| 2 | [Navigating the filesystem](02-navigating-the-filesystem.md) | Move anywhere in the filesystem tree and know what lives where |
| 3 | [Working with files](03-working-with-files.md) | Create, view, copy, move, and delete files and directories safely |
| 4 | [Getting help](04-getting-help.md) | Answer your own questions with man pages, --help, and friends |

## Before you start

You need a terminal on Ubuntu 26.04 (or any recent Ubuntu). Any of these works:

- A physical or virtual machine running Ubuntu 26.04 - open the **Terminal** app (`Ctrl+Alt+T`).
- Windows: [WSL2](https://learn.microsoft.com/en-us/windows/wsl/install) with the Ubuntu 26.04 distribution.
- A throwaway cloud VM or an LXD/Multipass container.

> [!TIP]
> Use a machine you're allowed to break. The exercises are safe, but the fastest way to learn Linux is to poke at things without fear.
