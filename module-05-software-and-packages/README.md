# Module 5 · Software & Packages

> **Goal:** own how software gets onto your machine - install, upgrade, inspect, and remove packages with apt, understand what a .deb actually is, and know when snap or building from source is the right call.

## Learning objectives

- Drive `apt`: update, upgrade, install, remove, autoremove, and search
- Understand repositories, `/etc/apt/sources.list.d/`, and package signing
- Inspect packages with `dpkg`: what's installed, which package owns a file
- Know how snaps differ from debs, and manage them with `snap`
- Handle Ubuntu release upgrades and unattended security updates
- Build a program from source once, so you know what packages save you from

## Planned lessons

1. apt essentials - the daily driver
2. Under the hood - dpkg, .deb files, and repositories
3. Snaps and other channels - snap, PPAs, and when to avoid them
4. From source - ./configure, make, and why packages exist
